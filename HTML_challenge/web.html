Summary: Latitude Vs. X



The purpose of this project was to analyze how weather changes as you get close to the equator. We first pulled data from the OpenWeatherMap API to assemble a dataset on on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various aspets of the weather vs. latitude. Factors we looked at included:Temperature,

cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
