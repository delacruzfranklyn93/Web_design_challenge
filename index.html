Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get close to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on on over 500 cities.

After assembling the dataset, we used matplotlib to plot various aspects of the weather vs. latitude. Factors we looked

at included: temperature, cloudiness, wind speed, and humidty. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descipitions of any trends and correlations witnessed
